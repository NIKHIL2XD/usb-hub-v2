# usb-hub-v2
A customized 4 port usb hub

## Parts Used

- SL2.1A board : C192893
- 1 USB 2.0 type A male : C319178
- 4 USB 2.0 type A female : C319178
- 1 Shotkey : C48192
- 7 10uF Capacitor : C19702
